三江项目目录结构
       
         一 公共文件
            1  CSS
                  |---base.css   （初始化样式）
                  |---common.css （公共样式 ）
                  |---public.css   (公共弹窗和提示信息样式，等等...)
                  |---swiper.min.css （轮播插件样式）
             
            2  JS
                  |---rem.js （pc端自适应方法）
                  |---jquery.min.js
                  |---fullplay.js  (全屏轮播方法)
                  |---common.js  （公共js，如：通用请求ajax）
                  |---public.js （公共js方法，如：时间转换，弹窗，提示信息）
                  |---pageMe.js （分页插件）
                  |---swiper.min.js （轮播插件）

           3  images （存放静态资源文件 如：图标）
           4  resources （存放临时资源文件 如：背景图）


         二 html结构文件

           1  登录 注册 绑定 网站主页
